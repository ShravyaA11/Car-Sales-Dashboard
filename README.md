# Car-Sales-Dashboard
This project presents an interactive Car Sales Dashboard developed in Power BI to analyze sales performance, monitor key business metrics, and generate actionable insights. The dashboard helps stakeholders track sales trends, evaluate vehicle performance, and make data-driven decisions.

# Objectives
* Analyze overall car sales performance.
* Track Year-to-Date (YTD) and Month-to-Date (MTD) sales.
* Compare current sales with previous year performance.
* Identify top-performing vehicle categories and regions.
* Provide interactive visualizations for business analysis.

# Key KPIs
* YTD Total Sales(Year-to-Date)
* MTD Total Sales(Month-to-Date)
* YOY Sales Growth(Year-on-Year)
* Average Selling Price
* Cars Sold
* Regional Sales Performance

# Dashboard Features
* Sales Overview Dashboard
* Weekly Sales Trend Analysis
* Sales by Body Style
* Sales by Vehicle Color
* Dealer Region Performance Map
* Company-Wise Sales Analysis
* Detailed Sales Information Table

# Tools & Technologies
* Power BI
* Power Query
* DAX
* Microsoft Excel

# Skills Demonstrated
* Data Cleaning & Transformation
* Data Modeling
* DAX Calculations
* KPI Development
* Business Intelligence
* Dashboard Design
* Data Visualization

# Conclusion
The dashboard provides a comprehensive view of car sales performance through interactive reports and visual analytics, enabling efficient business monitoring and decision-making.
